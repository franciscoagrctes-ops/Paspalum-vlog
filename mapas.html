File format

How the CSV should look

The map reads a comma-separated file with a header row and one row per occurrence. Column order doesn't matter, but the names do:

scientific_name,longitude,latitude
Paspalum guenoarum,-57.95,-30.90
Paspalum guenoarum,-58.15,-31.75
Paspalum guenoarum,-56.35,-32.85

If your data comes from Google Earth as KML/KMZ, or as a shapefile (.shp + .shx + .dbf + .prj zipped together), send it over and I'll convert it to this same format before you upload it here.
